Details

✔ Web application implemented using asp.net with C#, SQL Server, CSS, Bootstrap. This project is implemented basically to provide solution for people who are facing difficulties with civic issues like garbage dump, sewage leak, potholes etc. to register a complaint using images which are geotagged.
✔ Web application enables a transparent medium through which the users can register complaint, track the progress of their complaints and confirm the rectification of the issue on its completion.
